## 4. 가중치 산정 제안

### 4.1 기본 방식: PM 종 수 기반

- **가정:** 광물 종 수가 많은 PM일수록 해당 환경이 넓거나 접근이 쉬움.
  - PM47(풍화): 1,998종 — 표층·풍화 환경이 넓음
  - PM11(화산 분기구): 36종 — 국소적·희귀 환경
- **PM 가중치:** `weight(PM#) = species_count(PM#)` (또는 정규화)

### 4.2 조합 방식 (PM 종 수 + 지각 분포)

1. **PM 종 수 가중치**  
   `W_species(PM) = species_count(PM) / sum(species_count)`
2. **지각 분포 보정 계수** (대응되는 PM만)  
   `W_crust(PM) = mapped_crust_percentage` (예: PM19→0.25, PM40→0.30)
3. **최종 PM 가중치**  
   `W(PM) = α * W_species + (1-α) * W_crust` (α 예: 0.6~0.8)

### 4.3 광물별 뽑기 가중치

광물이 여러 PM에 출현할 수 있으므로:

```
mineral_weight = Σ W(PM_i)  for each PM_i the mineral appears in
```

또는 가중평균:

```
mineral_weight = (1 / n) * Σ W(PM_i)
```

## 5. ima_list.csv와의 매핑

- `ima_list.csv`의 Paragenetic Modes 열은 `PM47 - Low-T subaerial...` 형태.
- `PM47` → Hazen Table 1의 `#47`과 동일.
- 변환 시: `PM\d+` 정규식으로 PM 번호 추출 후, 위 가중치 표와 매칭.
